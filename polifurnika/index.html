📌 Bab 1

Apa itu JASP?

Pengantar, keunggulan, perbandingan, dan panduan instalasi JASP untuk pemula.

Tentang JASP

JASP (Jeffreys's Amazing Statistics Program) adalah perangkat lunak statistik yang gratis, open source, dan dikembangkan oleh Universitas Amsterdam. Didesain untuk ilmuwan sosial — cukup drag variabel, output muncul real-time dalam format APA. Bayesian dan Frequentist berdampingan dalam satu antarmuka.

✅ Keunggulan JASP
Gratis selamanya, antarmuka visual tanpa kode, output APA otomatis, Bayesian terintegrasi, R sudah bundled di dalamnya.
⚠️ Keterbatasan
Tidak mendukung .xlsx langsung (perlu konversi ke .csv), fitur lanjutan membutuhkan pemahaman R untuk kustomisasi lebih jauh.

Perbandingan dengan Software Lain

Perlu Kode✗ TidakYaSebagianYa

💡
Untuk pengajar dan tendik, JASP adalah pilihan terbaik: gratis, antarmuka visual, output langsung bisa disalin ke laporan Word atau PowerPoint tanpa format ulang.

Langkah Instalasi JASP

  • 1

    Buka jasp-stats.org/download/

    Pilih installer sesuai sistem operasi: Windows (.exe), macOS (.dmg), atau Linux (.AppImage).

  • 2

    Jalankan Installer

    Double-click file installer. Ikuti wizard instalasi. JASP tidak memerlukan R yang terpisah — sudah terintegrasi dalam paket instalasi.

  • 3

    Buka JASP & Cek Versi

    Buka JASP, pergi ke Menu (☰) → About JASP. Pastikan menggunakan versi terbaru (0.19+).

  • 4

    Atur Bahasa Indonesia

    ☰ → Preferences → Interface → Language → Bahasa Indonesia. Restart JASP agar perubahan berlaku.

🖥️ Bab 2

Memahami Antarmuka JASP

Kenali setiap elemen antarmuka JASP dan cara mengkustomisasinya.

Elemen Utama Antarmuka

☰ Hamburger Menu (kiri atas)

Buka/simpan file, ekspor hasil ke Word/PDF, koneksi OSF (Open Science Framework), dan Preferences.

⊞ Top Bar Analisis

Akses semua menu analisis: Descriptives, T-Tests, ANOVA, Regression, Frequencies, Factor, dsb.

✚ Plus Button (kanan atas)

Tambah modul analisis lanjutan: SEM, Meta-Analysis, Machine Learning, Network Analysis, dan lainnya.

⚙ Preferences

Kustomisasi tema, bahasa, desimal, resolusi grafik, dan nilai hilang. Akses via ☰ → Preferences.

📋
Tata letak layar: Panel kiri = Control Panel (pilih variabel & opsi analisis). Panel kanan = Output Panel (hasil analisis muncul otomatis, real-time saat opsi diubah).

Kustomisasi Preferences

📊

Data Preferences

Atur editor default, ambang batas kategori vs skala, dan kode missing value kustom (mis. 99, 999).

📋

Result Preferences

Tampilkan p-value tepat, jumlah desimal, resolusi grafik, dan latar transparan/putih untuk grafik.

🖥️

Interface Preferences

Pilih tema light/dark, bahasa (Indonesia tersedia), ukuran zoom, dan font untuk aksesibilitas.

✅
Pro Tip: Atur resolusi grafik ke 300+ DPI di Preferences → Results sebelum mulai analisis agar grafik siap publikasi tanpa perlu redo.

Manajemen Data

Format file, tipe variabel, dan dataset latihan Data_latihan_Furnitur.csv.

Format File yang Didukung JASP

.csv / .txt / .tsv .sav (SPSS) .dta (Stata) .ods .sas7bdat .xpt
⚠️
JASP tidak mendukung .xlsx secara langsung. Simpan dulu sebagai .csv dari Excel (Save As → CSV UTF-8), lalu buka di JASP.

Tipe Variabel di JASP

N

Nominal

Kategori tanpa urutan. Contoh: jenis kelamin, agama, etnis.
Simbol: ◉ oranye

O

Ordinal

Kategori berurutan. Contoh: tingkat pendidikan, skala Likert 1–5.
Simbol: ≡ biru

S

Scale / Kontinu

Angka terukur. Contoh: usia, skor nilai, pendapatan.
Simbol: ↗ hijau

💡
Tips: Klik ikon tipe variabel di header kolom untuk mengubahnya. JASP otomatis mendeteksi, tapi selalu verifikasi manual sebelum analisis.

2 · Komparasi dua kelompok independen

Walkthrough: Statistik Deskriptif

  • 1

    Buka Dataset

    ☰ → Open → pilih file Data_latihan_Furnitur.csv. Data muncul di panel kiri seperti spreadsheet.

  • 2

    Buka Descriptives

    Klik Descriptives di top bar → pilih Descriptive Statistics.

  • 3

    Drag Variabel

    Drag variabel Diameter_Log_cm, Kadar_Air_persen, Ketebalan_mm, Efisiensi_persen ke kotak Variables. Output langsung muncul real-time di panel kanan.

  • 4

    Centang Statistik

    Di panel kiri, centang: Mean, SD, Median, Minimum, Maximum, Skewness, Kurtosis.

  • 5

    Salin ke Word

    Klik kanan tabel di panel kanan → Copy. Paste ke Word — format APA otomatis dengan garis tabel yang tepat.

📈 Bab 5

Grafik, Tabel & Ekspor

Cara membuat grafik publikasi, kustomisasi, dan ekspor hasil ke Word/PDF.

Jenis Grafik di JASP

📊

Histogram & Bar

Via Descriptives → Plots → Distribution plots. Centang Display density untuk kurva normal overlay.

📦

Boxplot & Raincloud

Descriptives → Plots → Boxplots. Aktifkan Raincloud plot untuk visualisasi distribusi yang lebih kaya.

🔵

Scatter Plot

Regression → Correlation → Plots. Atau via Descriptives → Plots → Scatter plots untuk matrix scatter.

🌡️

Correlation Heatmap

Regression → Correlation → centang Heatmap. Visualisasi matriks korelasi antar semua variabel.

📉

Means Plots (ANOVA)

ANOVA → Plots → Marginal Means. Untuk visualisasi interaksi antar faktor dengan garis dan error bars.

🔬

Residual & QQ Plot

Regression → Plots → centang Residuals vs Fitted dan Q-Q Plot untuk cek asumsi normalitas residual.

💡
Tips Ekspor: Klik kanan grafik → Copy image atau Save image as. Untuk resolusi tinggi, atur ke 300+ DPI di ☰ → Preferences → Results sebelum analisis dimulai.

Cara Membuat & Menyalin Tabel

  • 1

    Jalankan Analisis

    Jalankan Descriptives atau analisis lainnya. Tabel output muncul otomatis di panel kanan.

  • 2

    Edit Label Langsung

    Double-click teks dalam tabel untuk mengubah label kolom atau judul tabel langsung di output JASP.

  • 3

    Klik Kanan → Copy

    Klik kanan tabel → Copy. Paste ke Word — format APA otomatis dipertahankan dengan garis tabel yang tepat.

  • 4

    Ekspor Semua Output

    ☰ → Export Results → Word/PDF/HTML untuk ekspor semua tabel & grafik dalam satu file laporan sekaligus.

📌
Gunakan ☰ → Export Results → Word untuk ekspor seluruh output analisis (semua tabel + grafik) dalam satu dokumen Word berformat rapi. Sangat berguna untuk menyusun laporan akhir.
💻 Bab 6

R for JASP

Akses kode R di balik setiap klik JASP — untuk reprodusibilitas, fleksibilitas, dan belajar R bertahap.

JASP sebenarnya berjalan di atas R — setiap klik di JASP menghasilkan kode R di belakang layar. Fitur R for JASP memungkinkan Anda melihat, menyalin, dan memodifikasi kode R tersebut untuk kontrol penuh atas analisis.

ℹ️
Anda tidak perlu instalasi R atau RStudio terpisah. R sudah terintegrasi dalam JASP. Fitur ini tersedia via Hamburger → R for JASP atau ikon </> di setiap output.

Mengapa Belajar R Sintaks di JASP?

🔁

Reprodusibilitas

Simpan skrip R untuk analisis yang dapat diulang kapan saja dengan dataset berbeda — lebih andal dari klik manual.

🎨

Fleksibilitas

Modifikasi grafik dan tabel beyond menu — warna kustom, anotasi, ukuran font, tema ggplot2.

📚

Belajar R Bertahap

JASP sebagai jembatan — lihat kode R otomatis, pelajari sintaksnya, kembangkan sendiri tanpa tekanan.

Cara Mengakses Kode R di JASP

  • 1

    Jalankan Analisis Apapun

    Misalnya: Descriptives → pilih variabel. Output muncul di panel kanan.

  • 2

    Klik Ikon </> di Output

    Di setiap tabel atau grafik, ada ikon kode di pojok kanan atas. Klik untuk melihat kode R-nya secara langsung.

  • 3

    Buka R Console di JASP

    ☰ → R for JASP. Muncul terminal R interaktif langsung dalam JASP — bisa mengetik perintah R.

  • 4

    Jalankan & Modifikasi

    Salin kode dari output, tempel ke R console, ubah parameter, dan jalankan untuk hasil kustom.

Contoh Kode R di JASP

1 · Statistik Deskriptif

R · Descriptives
# Statistik Deskriptif — setara klik JASP library(jaspDescriptives) # Baca data data <- read.csv("Data_latihan_Furnitur.csv") # Hitung deskriptif desc <- jaspDescriptives( data = data, vars = c("Diameter_Log_cm", "Kadar_Air_persen", "Efisiensi_persen"), mean = TRUE, sd = TRUE, median = TRUE, skew = TRUE ) print(desc)

2 · Histogram dengan ggplot2

R · ggplot2
# Histogram distribusi efisiensi furnitur library(ggplot2) data <- read.csv("Data_latihan_Furnitur.csv") ggplot(data, aes(x = Efisiensi_persen)) + geom_histogram( binwidth = 3, fill = "#0d7377", color = "white", alpha = 0.85 ) + labs( title = "Distribusi Efisiensi Penggunaan Bahan Baku", x = "Efisiensi (%)", y = "Frekuensi" ) + theme_minimal(base_size = 13)

3 · Independent T-Test

R · T-Test
# Independent samples t-test: Jenis Kayu vs Efisiensi data <- read.csv("Data_latihan_Furnitur.csv") # Filter hanya 2 grup (misal: Jati vs Pinus) data2 <- data[data$Jenis_Kayu %in% c("Jati", "Pinus"), ] data2$Jenis_Kayu <- factor(data2$Jenis_Kayu) hasil <- t.test(Efisiensi_persen ~ Jenis_Kayu, data = data2, var.equal = TRUE) print(hasil) # Hitung effect size (Cohen's d) library(effectsize) cohens_d(Efisiensi_persen ~ Jenis_Kayu, data = data2)

4 · Regresi Linear & Scatter Plot

R · Regression + Plot
# Multiple regression: prediktor → efisiensi furnitur data <- read.csv("Data_latihan_Furnitur.csv") model <- lm(Efisiensi_persen ~ Diameter_Log_cm + Kadar_Air_persen + Ketebalan_mm, data = data) summary(model) # Scatter: Kadar Air vs Efisiensi library(ggplot2) ggplot(data, aes(x = Kadar_Air_persen, y = Efisiensi_persen)) + geom_point(alpha = 0.4, color = "#0d7377") + geom_smooth(method = "lm", color = "#b45309", se = TRUE) + theme_minimal(base_size = 12)

5 · Boxplot per Grup

R · Boxplot
# Boxplot efisiensi berdasarkan jenis kayu library(ggplot2) data <- read.csv("Data_latihan_Furnitur.csv") data$Jenis_Kayu <- factor(data$Jenis_Kayu) ggplot(data, aes(x = Jenis_Kayu, y = Efisiensi_persen, fill = Jenis_Kayu)) + geom_boxplot(alpha = 0.7, width = 0.5) + geom_jitter(width = 0.15, alpha = 0.2, size = 0.8) + scale_fill_manual(values = c("#0d7377", "#b45309", "#5b21b6")) + labs( title = "Efisiensi berdasarkan Jenis Kayu", x = "Jenis Kayu", y = "Efisiensi (%)" ) + theme_minimal() + theme(legend.position = "none")
📝 Bab 7

Tugas Latihan Mandiri

Empat tugas praktik menggunakan Data Latihan Furnitur — dapat dikerjakan secara individu atau kelompok.

⚠️
Pastikan sudah mengunduh Data_latihan_Furnitur.csv dan JASP versi terbaru sebelum mengerjakan tugas. Semua analisis menggunakan dataset yang sama.

Tugas 1 — Statistik Deskriptif

📊

Statistik Deskriptif + Histogram

Hitung Mean, SD, Median, Skewness, Kurtosis untuk variabel Diameter_Log_cm, Kadar_Air_persen, Efisiensi_persen. Sertakan histogram dengan kurva distribusi normal.

Descriptives → Descriptive Statistics Plots → Distribution plots

Laporkan: Tabel deskriptif APA + 3 histogram. Tambahkan interpretasi singkat per variabel.

Tugas 2 — Uji T (Jenis Kayu)

⚖️

Apakah Efisiensi Berbeda antar Jenis Kayu (Jati vs Pinus)?

Uji apakah efisiensi penggunaan bahan baku (Efisiensi_persen) berbeda secara signifikan antara kayu Jati dan Pinus menggunakan Independent Samples T-Test.

T-Tests → Independent Samples T-Test Centang: Cohen's d, Descriptives

Laporkan: Nilai t, df, p-value, Cohen's d, dan interpretasi apakah perbedaan signifikan.

Tugas 3 — Analisis Korelasi

🔗

Korelasi antar Variabel Furnitur

Hitung matriks korelasi Pearson antara variabel Diameter_Log_cm, Kadar_Air_persen, Ketebalan_mm, Efisiensi_persen. Tampilkan heatmap korelasi.

Regression → Correlation Plots → Heatmap + Scatter matrix

Laporkan: Tabel korelasi APA + heatmap + interpretasi pasangan variabel yang paling kuat korelasinya.

Tugas 4 — Regresi Berganda

📈

Prediktor Efisiensi Furnitur

Prediksi variabel Efisiensi_persen menggunakan prediktor: Diameter_Log_cm + Kadar_Air_persen + Ketebalan_mm. Laporkan model regresi berganda.

Regression → Linear Regression Centang: R², F-statistic, Standardized β

Laporkan: Tabel koefisien APA (B, SE, β, t, p), nilai R² dan F, serta interpretasi prediktor signifikan.

✅
Format Laporan: Setiap tugas disajikan dengan (1) tabel output APA, (2) grafik relevan, dan (3) paragraf interpretasi singkat maksimal 3 kalimat. Ekspor menggunakan ☰ → Export Results → Word.
