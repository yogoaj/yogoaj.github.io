Selamat datang di repositori website resmi **Ruang Statistika**. Website ini berfungsi sebagai portofolio profesional, pusat publikasi ilmiah, dan platform layanan konsultasi statistik untuk mahasiswa (S1-S3) serta peneliti.

## 🚀 Fitur Utama
- **Interaktif R Shiny Dashboard:** Menampilkan visualisasi data kependudukan secara *real-time* yang terintegrasi langsung di halaman utama.
- **Service Hub:** Katalog layanan konsultasi (Uji Asumsi, Regresi, SEM/PLS, Bayesian) dengan skema harga transparan.
- **Academic Portfolio:** Daftar publikasi jurnal terindeks (Springer, Jurnal STIS, dll) dan buku proyeksi penduduk.
- **Legacy Archives:** Dokumentasi proyek-proyek terdahulu yang bermigrasi dari Google Sites.

## 🛠️ Stack Teknologi
- **Frontend:** HTML5, CSS3 (Modern UI dengan DM Sans & DM Serif Display), JavaScript.
- **Data Engine:** R Shiny (Hosted on Shinyapps.io).
- **Deployment:** GitHub Pages.
- **Design:** Clean & Minimalist Professional Layout.
