📋 Bab 1 — Pengantar

Tiga Pertanyaan Sebelum Memilih Uji Statistik

Sebelum buka SPSS, Python, atau aplikasi apapun, Anda harus menjawab tiga pertanyaan ini. Jawaban dari ketiganya akan menentukan uji statistik yang tepat secara otomatis.

1.1 Apa skala pengukuran variabel saya?

Skala data menentukan operasi matematika apa yang boleh dilakukan pada data tersebut. Ada empat jenis:

🔢 Nominal

Kategori tanpa urutan. Contoh: jenis kelamin (laki/perempuan), agama, jurusan kuliah. Tidak bisa dijumlahkan atau diurutkan secara bermakna.

📊 Ordinal

Kategori dengan urutan, tapi jarak antar kategori tidak sama. Contoh: tingkat kepuasan (tidak puas, cukup, puas, sangat puas), skala Likert 1–5.

📏 Interval

Angka dengan jarak yang sama, tapi tidak punya titik nol absolut. Contoh: suhu Celsius, skor IQ. Bisa dijumlahkan, tapi perbandingan rasio tidak bermakna.

⚖️ Rasio

Angka dengan jarak sama DAN punya titik nol absolut. Contoh: pendapatan, berat badan, usia. Bisa semua operasi matematika. Ini level data tertinggi.

1.2 Berapa kelompok yang saya bandingkan?

Pertanyaan ini berlaku saat tujuan Anda adalah membandingkan atau melihat perbedaan antar kelompok:

  • 2 kelompok independen → t-test atau Mann-Whitney
  • 2 kelompok berpasangan (sebelum-sesudah) → Paired t-test atau Wilcoxon
  • 3 kelompok atau lebih → ANOVA atau Kruskal-Wallis

Jika tujuan Anda adalah melihat hubungan atau pengaruh (bukan perbedaan), langsung ke pertanyaan berikutnya.

1.3 Apakah data saya terdistribusi normal?

Ini pertanyaan paling sering salah dipahami. Normalitas bukan syarat mutlak untuk semua uji statistik — hanya untuk uji parametrik.

💡 Aturan Praktis
Jika N ≥ 30 (Central Limit Theorem): distribusi rata-rata sampel mendekati normal meskipun data aslinya tidak normal. Tetap lakukan uji normalitas, tapi jangan panik jika hasilnya tidak normal — ada alternatif nonparametrik.

Cara uji normalitas yang umum:

  • Kolmogorov-Smirnov (KS-test) — untuk N besar (> 50)
  • Shapiro-Wilk — lebih akurat untuk N kecil (≤ 50)
  • Q-Q Plot — visualisasi, data normal jika titik mengikuti garis diagonal

🎯 Bab 3 — Skenario

Panduan Skenario Penelitian Sosial

Skenario nyata yang paling sering ditemui dalam penelitian skripsi dan tesis di bidang sosial, manajemen, psikologi, dan pendidikan.

A

Pengaruh Variabel X terhadap Y

Contoh: "Apakah motivasi kerja berpengaruh terhadap kinerja karyawan?"

  • Jika X dan Y keduanya numerik kontinu → Regresi Linier (OLS)
  • Jika Y biner (ya/tidak, lulus/tidak lulus) → Regresi Logistik
  • Jika ada variabel mediasi (M) di tengah jalur X→Y → Analisis Mediasi
  • Jika pengaruh X terhadap Y tergantung variabel Z → Analisis Moderasi
B

Perbedaan Antar Kelompok

Contoh: "Apakah ada perbedaan kepuasan kerja antara karyawan tetap dan kontrak?"

  • 2 kelompok, data normal, varians sama → Independent t-test
  • 2 kelompok, data normal, varians berbeda → Welch's t-test
  • 2 kelompok, data tidak normal atau N kecil → Mann-Whitney U
  • 3+ kelompok, data normal → One-way ANOVA + post-hoc Tukey HSD
  • 3+ kelompok, data tidak normal → Kruskal-Wallis + post-hoc Dunn
C

Hubungan Antar Variabel

Contoh: "Apakah ada hubungan antara usia dan produktivitas kerja?"

  • Dua variabel interval/rasio, normal → Pearson Correlation (r)
  • Minimal salah satu ordinal, atau tidak normal → Spearman Correlation (ρ)
  • Dua variabel nominal → Chi-Square Test of Association (χ²)
D

Sebelum dan Sesudah (Pre-Post)

Contoh: "Apakah pelatihan meningkatkan kompetensi karyawan?"

  • Data numerik, distribusi normal → Paired t-test
  • Data numerik, tidak normal atau ordinal → Wilcoxon Signed-Rank Test
  • Data kategori (proporsi berubah?) → McNemar Test
✅ Tips untuk Bab Metodologi
Selalu sebutkan: (1) nama uji yang digunakan, (2) alasan memilih uji tersebut berdasarkan skala data dan distribusi, (3) perangkat lunak yang digunakan (SPSS, Python, R, dll.).
📝 Bab 4 — Pelaporan APA

Cara Melaporkan Hasil Statistik (APA 7th)

Setelah memilih uji yang tepat dan menjalankan analisis, langkah berikutnya adalah melaporkan hasilnya dengan benar. Berikut format APA 7th yang paling umum untuk setiap uji.

Independent t-test
Format APA
Hasil uji independent samples t-test menunjukkan terdapat perbedaan yang signifikan antara kelompok A (M = 4.21, SD = 0.63) dan kelompok B (M = 3.87, SD = 0.71), t(98) = 2.54, p = .013, d = 0.51 [efek sedang].
One-way ANOVA
Format APA
Terdapat perbedaan yang signifikan pada variabel kinerja antar tiga kelompok jabatan, F(2, 87) = 8.34, p < .001, η² = .16. Uji post-hoc Tukey HSD menunjukkan kelompok manajer berbeda signifikan dari staf (p = .002).
Pearson Correlation
Format APA
Terdapat korelasi positif yang signifikan antara motivasi dan kinerja, r(98) = .67, p < .001. Artinya, semakin tinggi motivasi, cenderung semakin tinggi kinerja karyawan.
Regresi Linier OLS
Format APA
Model regresi secara keseluruhan signifikan, F(3, 96) = 17.42, p < .001, R² = .35. Variabel motivasi secara signifikan memprediksi kinerja, β = .42, t = 4.81, p < .001.
Chi-Square
Format APA
Uji chi-square menunjukkan terdapat hubungan yang signifikan antara jenis kelamin dan pilihan karier, χ²(1) = 5.83, p = .016, φ = .24.
