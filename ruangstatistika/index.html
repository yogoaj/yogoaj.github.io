FAQ

Pertanyaan yang Sering Ditanyakan

Apa bedanya dengan pakai ChatGPT langsung?
ChatGPT tidak bisa membaca file CSV/Excel Anda, tidak menghasilkan uji statistik yang akurat, dan tidak tahu standar penulisan APA 7th. Ruang Statistika mengintegrasikan library statistik scikit-learn, scipy, statsmodels, dan semopy yang teruji secara ilmiah, lalu mengirim hasilnya ke AI untuk diinterpretasikan dalam bahasa akademis Indonesia yang tepat. Ini kolaborasi terbaik antara presisi statistik dan kecerdasan bahasa.
Berapa biaya API Key untuk AI-nya?
Groq (Llama 3.3 70B) sepenuhnya gratis untuk penggunaan normal — tidak perlu mengeluarkan biaya apapun. Google Gemini memiliki tier gratis yang generous. Anthropic Claude dan OpenAI GPT tersedia dengan bayar sesuai penggunaan. Untuk skripsi/tesis, biaya API biasanya di bawah Rp 10.000 total. Anda tidak berbagi kuota dengan pengguna lain — itu keuntungan model "Private by Design" kami.
Apakah hasil statistiknya akurat untuk publikasi jurnal?
Ya. Kami menggunakan library Python standar industri: scipy untuk uji statistik dasar, statsmodels untuk regresi OLS, diagnostik dan RLM robust, pingouin untuk ANOVA dan effect size, dan semopy untuk SEM/CFA. Semua library ini adalah standar yang sama digunakan peneliti di seluruh dunia dan menghasilkan hasil identik dengan SPSS, R, dan AMOS untuk data yang sama.
Apakah versi gratis benar-benar bisa dipakai tanpa bayar?
Ya, sepenuhnya. Modul analisis dasar (deskriptif, validitas, korelasi, uji beda, nonparametrik, regresi/ANOVA/logistik dasar, EDA, compute variabel) tersedia tanpa License Key. Sejak v4.5, pengguna Free juga bisa generate laporan .docx atau .md gratis 1 kali per hari (tanpa narasi AI). License Pro hanya diperlukan untuk fitur lanjutan: laporan tak terbatas, narasi AI, grafik tertanam, dan modul Pro seperti SEM, mediasi, moderasi.
Bagaimana cara aktivasi lisensi setelah pembayaran?
Setelah konfirmasi pembayaran, Anda akan menerima kode lisensi unik dalam 1×24 jam via WhatsApp atau email. Masukkan kode tersebut di sidebar aplikasi, dan akses Pro Anda langsung aktif sesuai paket yang dipilih.
Apakah ada garansi uang kembali?
Ya, kami memberikan garansi 7 hari tanpa syarat. Jika dalam 7 hari Anda merasa Ruang Statistika tidak sesuai ekspektasi, hubungi kami dan kami akan proses refund penuh. Kami yakin dengan kualitas produk kami.
Apakah Paket Starter (1 bulan) bisa diperpanjang?
Tentu saja. Anda bisa upgrade ke Paket Premium (6 bulan) atau Professional (1 tahun) kapan saja — bahkan sebelum masa aktif habis — dan hanya perlu membayar selisih harganya. Hubungi kami untuk mendapatkan harga upgrade terbaik.

Google OAuth dihapus agar setup lebih sederhana. Session token Supabase disimpan di session_state dan di-restore otomatis saat refresh.
